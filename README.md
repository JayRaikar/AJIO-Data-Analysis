# AJIO-Data-Analysis
This project focuses on analyzing AJIO’s e-commerce data to derive insights on sales performance, customer behavior, product trends, profit distribution, and return impacts. The analysis was performed using Excel for data cleaning, SQL for data exploration, and Power BI for dashboard reporting.

Project Objectives:
Understand AJIO’s sales performance & revenue distribution.
Identify high-performing products, categories, and locations.
Analyze customer purchase patterns and repeat order behavior.
Evaluate return rates and their impact on overall revenue.
Build an interactive dashboard for KPI monitoring & business decisions.

Tools & Technologies Used:
Tool	           Purpose
Excel	         Data cleaning, preprocessing, and formatting.
SQL (MySQL)	   Querying, data extraction, aggregations, KPIs.
Power BI	     Data modeling, visualizations, dashboards.
DAX Measures   KPI calculations & dynamic reporting.
